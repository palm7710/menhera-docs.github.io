# Error Handling

状態が崩れたときは、まずエラー種別を見分けます。

## Fatal Error

感情爆発に相当します。

- 長文メッセージ
- 着信連打
- 被害妄想
- 強い断定

## Silent Failure

沈黙系の故障です。  
外からは静かに見えますが、内部では不安定です。

- 急に冷たくなる
- 無視
- `大丈夫` だけ残して閉じる

## 対応の基本

| Error | 対応 |
| --- | --- |
| `返信遅延` | 返せないなら目安を先に出す |
| `既読無視` | 既読だけで止めない |
| `SNS監視` | 返信より先に更新するなら一言入れる |
| `感情爆発` | 論破せず、入力を減らして落ち着かせる |
| `沈黙` | 放置せず、低刺激で生存確認する |
