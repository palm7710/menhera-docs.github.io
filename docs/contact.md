# 他者との関わり方

人との距離は、近さより解像度で決まります。
ここでは「仲良くする方法」ではなく、「壊れにくく関わるための条件」を明文化します。

## してほしいこと

- 予定変更は短くてもいいので早めに伝える
- 返信が難しい時は猶予をもらえると助かる
- 長い議論の前に、いま話せる状態か確認してほしい
- 励ましよりも、事実確認を優先してほしい時がある

## してほしくないこと

- 返答の遅さを人格評価に直結させること
- 感情が揺れている時に結論を急がせること
- 曖昧な優しさで期待だけを残すこと
- 共有した弱さを別の場面で材料にすること

## 会話時の注意点

困っている時ほど、複雑な比喩や含みのある表現を処理しづらくなります。
確認可能な言葉と、短い文章のほうが助かる場面が多いです。

```txt
better:
  - "今すぐ返事は必要ですか"
  - "今日は会わない方がいいですか"
  - "手伝えることを一つだけ言うと何ですか"

avoid:
  - "普通はこうするよね"
  - "気にしすぎじゃない?"
  - "落ち着いたら分かるよ"
```

## 渡しやすい共有文

必要なら、次のような短文をそのまま渡せる形で使えます。

```md
今は刺激に弱く、長い会話が難しい状態です。
急ぎでなければ返答を待ってください。
必要な連絡は、短く具体的に送ってもらえると助かります。
```
