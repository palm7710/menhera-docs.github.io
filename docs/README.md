## このサイトについて

ここは、気分や反応を「雰囲気」ではなく「条件」として書き下ろすための場所です。  
不調を美化せず、否定もせず、観測可能な現象として扱います。

最初は [getting-started](/getting-started/) から読むと全体像が崩れません。  
急いでいる場合は [contact](/contact/) を先に開き、他者へ渡すための要点だけ確認してください。
