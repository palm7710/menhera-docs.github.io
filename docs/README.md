## このサイトについて

このサイトは `menhera.js` という架空のライブラリのドキュメントです。  
テーマは「メンヘラ的な揺らぎを、状態管理とイベント駆動の比喩で読む」です。

最初は [getting-started](/getting-started/) で導入を読み、次に [core-model](/core-model/) と [state-machine](/state-machine/) を見ると全体像が掴みやすくなります。  
急いでいる場合は [error-handling](/error-handling/) と [coping](/coping/) から先に見ても構いません。
