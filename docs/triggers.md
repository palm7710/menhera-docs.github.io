# Triggers / Behaviors

`Trigger` はきっかけで、`Behavior` はその結果出てくる振る舞いです。  
ここではよくある組み合わせをまとめます。

## 返信遅延

`timeout` に相当します。  
問題は時間そのものより、「返事が来る見込みが見えないこと」です。

### 典型的な挙動

- 深読み
- 連投
- `今なにしてるの？`
- SNS の監視強化

## 既読無視

返信遅延よりさらに強いイベントとして扱われがちです。  
`見たのに返していない` という解釈が乗るためです。

### 典型的な挙動

- 過去ログ分析
- 温度差の比較
- 急に冷たくなる
- 感情爆発または沈黙

## SNSシグナル

返信はないのに、SNS だけ更新されている状態です。  
`watchSNS()` が最も活発になる条件です。

### 典型的な挙動

- ストーリーの閲覧時刻確認
- 他者との交流チェック
- `私には返してないのに` という比較

## 愛情不足

愛情が足りないというより、足りているか確認できない状態です。

### 典型的な挙動

- お姫様モード
- 試し行動
- 優先順位の確認
- `ほんとに好き？` の再問い合わせ

## 比較

比較対象は他人だけではありません。  
過去の自分、昔の会話、前の態度も比較材料になります。

### 典型的な挙動

- `前はもっと優しかった`
- 過去ログ分析
- 元恋人や他人との比較
- 自己否定の再起動
