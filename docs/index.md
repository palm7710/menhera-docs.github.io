# menhera-docs

感情は、整理されていないままでは誤作動しやすい。
このサイトは、揺れやすい心の挙動を技術文書のように記述するための静かな保管庫です。

## このサイトでできること

- 状態の輪郭を言語化する
- 引き金になりやすい条件を整理する
- 崩れたときの応急処置を事前に用意する
- 他者に渡せる最低限の取り扱い説明をまとめる

## 主要ページ

- [最初に読む](/getting-started): 読み方、前提、注意点
- [典型的な状態](/symptoms): よく出る反応と内側の変化
- [トリガー整理](/triggers): 負荷が跳ね上がる条件の棚卸し
- [対処法](/coping): 平常時と緊急時の手順
- [他者との関わり方](/contact): 伝達したいことの雛形

## 使い方の雰囲気

このドキュメントは診断書でも、誰かを説得するための武器でもありません。
自分の状態を無理なく観測し、必要な断面だけを取り出して使うための設計メモです。

```txt
status: unstable
priority: low-stimulation
recommended_action:
  - reduce notifications
  - postpone difficult replies
  - switch to short sentences
```
