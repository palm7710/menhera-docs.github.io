# Getting Started

このページは、`menhera-docs` の前提をそろえるための入口です。
全部を一度に読まなくても構いません。必要な章だけを参照し、今の自分に合う粒度で使ってください。

## このドキュメントの読み方

状態は固定値ではなく、時間帯や相手や疲労で変わります。
そのため各ページは「完全な定義」ではなく、「再確認しやすい記録」として設計しています。

### おすすめの進め方

1. まずは [典型的な状態](/symptoms) を読み、自分に近い反応を拾う
2. 次に [トリガー整理](/triggers) で悪化条件を言葉にする
3. 最後に [対処法](/coping) と [他者との関わり方](/contact) を必要に応じて更新する

## 想定読者

- 感情の波をあとから振り返るための書式がほしい人
- しんどい時に長文で説明できず、短い共有文を準備したい人
- 周囲とのすれ違いを減らすため、境界線を明文化したい人

### 向いていない使い方

- 自分を単一のラベルに固定すること
- 他人を管理するためのルール集にすること
- 緊急の医療判断をこの文書だけで代替すること

## 注意事項

このサイトはケアの補助線であり、専門家の支援の代替ではありません。
希死念慮や自傷衝動が強い場合、ひとりで整えようとせず、地域の相談先や医療機関、信頼できる人に接続してください。
