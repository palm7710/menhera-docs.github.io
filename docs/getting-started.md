# はじめに

この文書は、感情を共感で読むためのものではなく、取り扱い条件を確認するためのものです。  
読み手が本人でも他者でも、まずは「何が起きるか」「何を避けるか」「どう連絡するか」の順で把握してください。

## このドキュメントの読み方

各ページは、ひとつの問題を切り分けて扱います。  
状態は [典型的な状態](/symptoms)、起点は [トリガー](/triggers)、応答手順は [対処法](/coping)、対人連絡は [他者との連携](/contact) に分離されています。

最初から順に読む必要はありません。現在の状況に近い項目だけ拾っても意味が通るように設計しています。  
ただし、急いで判断するときほど、単発の症状だけではなく前後の条件も確認したほうが誤差が減ります。

## 想定読者

- 自分の不調を言語化したい本人
- 本人から説明を受ける家族、友人、同僚
- すぐに結論を出さず、観察から入りたい支援者

読むときに必要なのは、深い理解よりも低いノイズです。  
正しさより再現性を優先し、感じ方より対処可能性を見る姿勢を推奨します。

## 注意事項

このサイトは診断書ではありません。  
医療的判断、緊急対応、法的判断の代替には使えません。

また、書かれている現象は常に一定ではありません。  
同じ刺激でも、睡眠不足、空腹、連続した予定、ひとつ前の会話などで反応は変わります。

> 読み手に求めるのは、正解を当てることではなく、余計な刺激を増やさないことです。
