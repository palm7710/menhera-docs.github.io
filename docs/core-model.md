# Core Model

`menhera.js` の中心は、感情を雰囲気で読むのではなく、状態と入力と出力に分けて扱うことです。

## State

`State` は、感情や精神状態の内部ステータスです。  
平たく言えば「今どれくらい安定しているか」です。

| State | 説明 |
| --- | --- |
| `Stable` | 安定状態 |
| `Anxious` | 不安や疑念が増加している状態 |
| `Dependent` | 強い依存状態 |
| `Unstable` | 感情の波が激しい状態 |
| `Exploding` | 感情爆発状態 |
| `SelfDestructive` | 自己破壊傾向が強い状態 |

## Trigger

`Trigger` は、状態変化を起こす要因です。  
日常語で言えば「しんどくなるきっかけ」です。

| Trigger | 内容 |
| --- | --- |
| 返信遅延 | 既読無視、未返信、返答の見通しがない状態 |
| SNS検知 | 他人との交流、投稿頻度、アイコン変更など |
| 愛情不足 | 会う頻度の減少、態度の変化、優先順位の低下 |
| 孤独 | 一人の時間が長い、話し相手がいない |
| 比較 | 他人と自分、過去の自分、元恋人などとの比較 |

## Behavior

`Behavior` は、状態に応じて外に現れる振る舞いです。  
つまり「周囲から見える動作」です。

| Behavior | 内容 |
| --- | --- |
| 連投連絡 | 頻繁な LINE、通話要求、確認の反復 |
| 感情爆発 | 怒り、泣き、長文送信、連投 |
| 試し行動 | わざと冷たくする、返信を遅らせる |
| SNS吐露 | 病み投稿、意味深、ポエム化 |
| 監視行動 | 行動把握、最終ログイン、更新状況の確認 |
| 関係リセット | 衝動的な遮断、ブロック、全部終わり宣言 |

## モデルの考え方

この三要素はこう繋がります。

```txt
Trigger が入る
-> State が変わる
-> Behavior として出力される
```

## Feature Specification

ここでは、よく話題にされる特徴を仕様っぽく整理します。

### Personality Traits

| Feature | 説明 |
| --- | --- |
| かまってちゃん | 承認欲求が強く、反応の有無を重く見る |
| 自己肯定感の低さ | `どうせ自分なんて` が初期値になりやすい |
| 寂しがり屋 | 孤独耐性が低く、無音時間に弱い |
| 繊細 | 小さな変化に敏感 |
| ネガティブ思考 | 良い可能性より悪い可能性を先に読む |

### Relationship Behavior

| Feature | 説明 |
| --- | --- |
| 頻繁な連絡 | 常時接続を求めやすい |
| 浮気疑い | 信頼より確認が先に立つ |
| 独占欲 | 他者との関係を制限したくなる |
| 過剰な尽くし | 愛情を一気に過投入する |
| 監視行動 | 行動や返信状況を把握したくなる |

### Emotional Patterns

| Feature | 説明 |
| --- | --- |
| 感情の起伏が激しい | 急な上下が起きる |
| 極端な愛情表現 | 好きと嫌いの振れ幅が大きい |
| 見捨てられ不安 | 関係崩壊を常に予測しやすい |
| 自己否定 | 自己評価が低く、褒めを保持しづらい |

### Behavioral Patterns

| Feature | 説明 |
| --- | --- |
| SNS病み投稿 | 間接的な SOS として出やすい |
| 人間関係リセット | 衝動的に関係を切りたくなる |
| 試し行動 | 愛情確認を行動で試す |
| 夜行性 | 生活リズムの崩れが状態に影響する |

### Background Factors

| Factor | 説明 |
| --- | --- |
| トラウマ | 過去の傷が現行の反応に影響する |
| 家庭環境 | 愛情不足や安心感の基盤不足につながる |
| 友人関係の少なさ | 孤立傾向が依存を強めやすい |
