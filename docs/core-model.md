# Core Model

`menhera.js` の中心は、感情を雰囲気で読むのではなく、状態と入力と出力に分けて扱うことです。

## State

`State` は、感情や精神状態の内部ステータスです。  
平たく言えば「今どれくらい安定しているか」です。

| State | 説明 |
| --- | --- |
| `Stable` | 落ち着いている状態 |
| `Anxious` | 不安や疑念が増えている状態 |
| `Dependent` | 特定対象への依存が強い状態 |
| `Unstable` | 感情の波が激しい状態 |
| `SelfDestructive` | 自己否定が強く、生活が荒れやすい状態 |
| `Exploding` | 感情が外に爆発している状態 |

## Trigger

`Trigger` は、状態変化を起こす要因です。  
日常語で言えば「しんどくなるきっかけ」です。

| Trigger | 内容 |
| --- | --- |
| 返信遅延 | 既読無視、未返信、返答の見通しがない状態 |
| SNSシグナル | 他者との交流、投稿頻度、アイコン変更など |
| 愛情不足 | 会う頻度の減少、態度の変化、優先順位の低下 |
| 孤独 | 一人の時間が長い、話し相手がいない |
| 比較 | 他人と自分、過去の自分、元恋人などとの比較 |

## Behavior

`Behavior` は、状態に応じて外に現れる振る舞いです。  
つまり「周囲から見える動作」です。

| Behavior | 内容 |
| --- | --- |
| 感情爆発 | 怒り、泣き、長文送信、連投 |
| 試し行動 | わざと冷たくする、返信を遅らせる |
| SNS吐露 | 病み投稿、意味深、ポエム化 |
| 依存行動 | 連投 LINE、通話要求、確認の反復 |
| 自己破壊寄り行動 | 睡眠崩壊、食欲の乱れ、生活管理の破綻 |

## モデルの考え方

この三要素はこう繋がります。

```txt
Trigger が入る
-> State が変わる
-> Behavior として出力される
```
