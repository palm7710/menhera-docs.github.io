# 典型的な状態

ここでは、よく現れる反応を「症状」ではなく運用上の状態として整理します。
良し悪しよりも、どの条件でどう変化するかを観測することを優先します。

## 思考が詰まる

言葉を選ぶまでに時間がかかり、返信や判断が極端に遅くなる状態です。
外からは無視や怠慢に見えても、内側では処理待ちが積み上がっています。

### よくある兆候

- 短い返答すら送れない
- 選択肢が多いほど停止する
- 既読をつけた後に罪悪感が増える

## 刺激に過敏になる

音、通知、予定変更、相手の語気など、普段は流せるものが鋭く刺さる状態です。
判断力より先に防御反応が立ち上がるため、説明より遮断が必要なことがあります。

### 反応の例

- 通知音で心拍が上がる
- 冗談を攻撃として受け取りやすい
- 明るい場所や雑音が続くと疲労が急増する

## 反芻が止まらない

過去の会話や失敗を繰り返し再生し、現在の作業に戻れなくなる状態です。
解決のために考えているつもりでも、実際には消耗だけが増えることがあります。

```txt
loop:
  input: "あの言い方はまずかったかもしれない"
  branch_a: "嫌われたかもしれない"
  branch_b: "説明し直すべきかもしれない"
  result: "寝るまで同じ話題が続く"
```

## 急に平気なふりをする

負荷が高いほど、逆に明るく整って見せようとすることがあります。
この状態は一見安定して見えますが、その後に反動が来やすい点に注意が必要です。
