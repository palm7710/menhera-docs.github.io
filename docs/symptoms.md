# Signals / Symptoms

ここでは、内側の思考パターンと、外側から見える変化を整理します。

## Cognitive Patterns

### 極端思考

少しの違和感を、そのまま最悪の結論に接続しやすい傾向です。  
たとえば、短文の返信ひとつで「嫌われた」と解釈するような動きです。

### 見捨てられ不安

関係の崩壊を常に予測してしまう状態です。  
小さな変化にも過剰反応しやすく、返信遅延や予定変更が大きく見えます。

### 自己否定

`私なんて` がデフォルトで起動している状態です。  
褒められても信用できず、よい出来事より悪い予兆を優先して読みます。

## Physical Effects

感情の不安定さは、しばしば身体にも出ます。

- 頭痛
- 胃痛
- 不眠
- 体調不良

ここで言いたいのは医療的な説明ではなく、精神状態が身体症状っぽく見えることがある、という観測です。

## External Signals

外部から見える変化もあります。

| Category | 内容 |
| --- | --- |
| 外見 | 肌荒れ、傷跡、露出や雰囲気の変化などが話題化されやすい |
| 行動 | SNS 多用、夜型、アイコン変更、過去ログ分析の反復 |

## 関係性のサイン

- 既読無視に強く反応する
- 返信遅延を重く受け取る
- 深読みが止まらない
- `仕事と私どっち` 問題に入りやすい
